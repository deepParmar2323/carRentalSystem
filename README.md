# 🚗 Car Rental System (Java)

A simple **console-based Car Rental System** developed using **Java** and **Object-Oriented Programming (OOP)** concepts.
This project allows users to rent and return cars while managing customer and rental details.

---

## 📌 Features

* 🚘 View available cars
* 👤 Add new customers
* 📅 Rent a car for specific days
* 💰 Automatic price calculation
* 🔄 Return rented cars
* 📋 Simple menu-driven interface

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Encapsulation, Classes, Objects)
* Collections (`ArrayList`)
* Scanner (User Input)

## ⚙️ How It Works

1. User selects an option from the menu:

   * Rent a Car
   * Return a Car
   * Exit

2. For renting:

   * Enter name
   * Choose available car
   * Enter number of days
   * Confirm booking

3. For returning:

   * Enter car ID
   * System updates availability

---

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/car-rental-system.git
   ```

2. Open in any Java IDE (IntelliJ / Eclipse / VS Code)

3. Compile and run:

   ```
   javac CarRentalSystem.java
   java CarRentalSystem
   ```
